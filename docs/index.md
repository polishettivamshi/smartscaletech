# SmartScaleTech

**SmartScaleTech** is a high-performance Digital Branding Agency helping businesses and job seekers create a powerful digital presence.

## Services

### Career Services
- **ATS Resume Writing** – Professionally crafted resumes optimized for Applicant Tracking Systems
- **LinkedIn Optimization** – Profile enhancement to maximize recruiter visibility

### Technical Services
- **UI/UX Design** – Clean, modern interface design for web and mobile
- **Web Development** – Scalable, performant web applications

### Marketing Services
- **Digital Branding** – Build a consistent and compelling brand identity
- **Content Strategy** – Targeted content to grow your audience

## Tech Stack

The platform is built using:
- **React 19** + **Vite** – Modern front-end framework
- **Tailwind CSS 4** – Utility-first responsive styling
- **Framer Motion** – Smooth UI animations
- **TypeScript** – Type-safe development

## Getting Started

Visit the live site or clone the repository and follow the setup instructions in the README to run locally.
